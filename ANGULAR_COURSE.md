## Course Description

Master Angular 20 from the ground up and build awesome, reactive web apps. This comprehensive course covers everything from basics to advanced topics including Components, Directives, Services, Pipes, HTTP access, Authentication, Routing, Modules, NgRx, Testing, and much more!

## Prerequisites

- Basic knowledge of HTML, CSS, and JavaScript
- No Angular or TypeScript knowledge required
- A modern code editor (VS Code recommended)
- Node.js and npm installed on your machine

## Learning Outcomes

After completing this course, you will be able to:

✓ Develop modern, complex, responsive and scalable web applications with Angular  
✓ Establish yourself as a frontend developer with deep Angular fundamentals  
✓ Fully understand the architecture of Angular applications  
✓ Create single-page applications with Angular  
✓ Use TypeScript to write robust code  
✓ Master Components, Directives, Services, Pipes, HTTP, Authentication, and more

## How to Load the Course Data

The comprehensive Angular course is included in the seed data. To populate your database:

1. Ensure PostgreSQL is running
2. Navigate to the backend directory:
   ```bash
   cd backend
   ```

3. Run the seed script:
   ```bash
   npm run seed
   ```

This will create:
- 4 test users (admin, 2 students, 1 instructor)
- The complete Angular course with 15 sections, 100+ lessons, and 6 quizzes
